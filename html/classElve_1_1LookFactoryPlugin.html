Detailed Description

The LookFactoryPlugin abstract class.

This QPlugin interface allow to replace the way the graph's node are drawn in the viewport. This class use the abstract factory pattern. Meaning that when re-implementing it's sole role will be to instantiate subclasses of abstract objects.

-NodeLooks for the nodes -EdgeLooks for the edges
